
# Baccarat
## Classic table game implemented in the best language ever created, C 👍

## NOTE
You will need to have [raylib](https://github.com/raysan5/raylib) dev files installed in order to compile the source code 

## Version
This version of baccarat is based on *[Punto Banco](https://en.wikipedia.org/wiki/Baccarat#Versions)* with some modifications. Mainly, betting on *Banker* does not charge a 5% fee 😂

## Game
The game starts with $20 and "ends" when $0. There is no *end game* state. To replay, just close and run the game again

## Purpose
This game was made with the intention of just building something simple for fun and there are a lot of rough edges. However, if I wait until everything is polished and perfect then the game would probably never see the light of day 😂
